No installation required! First unzip all the files into a directory of your choosing, such as C:\users\{username}\AppData\Local\Programs\DefectPX, or C:\Portable\DefectPX. (Don't use a directory that requires Administrator privileges, such as a subdirectory under Program Files). Then locate the main application executable file (DefectPX.exe) and launch it.

To uninstall, just delete the extracted files, the portable version does not make any changes to your system.
